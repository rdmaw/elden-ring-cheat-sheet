FAQ

How do I use the cheat sheet?

This project is a lightweight collection of checklist sheets, covering everything from single item pickups, NPCs, questlines, bosses, achievements and much more. If this is your first playthrough, I recommend that you complete the game on your own first, then use the cheat sheets for your later playthroughs instead.

Open the sheets menu and select a sheet to get started

Helpful info about soft caps, upgrade tables, rune item values and more can be found in the Resources sheet.

In the Walkthrough sheet, you'll find a step-by-step checklist guide through the entire game, covering as much as possible in a single playthrough. If you're not interested in a guided approach, but still want to make sure you don't miss any important questlines or bosses, try the NPC Walkthrough or Bosses sheets instead.

The Achievements sheet can be used to track achievements and legendary equipment.

Similarly, the Armaments, Armor, Ashes, and Spells sheets can be used to track items in their respective categories. For tracking merchant inventories, use the Merchants sheet.

For miscellaneous items, such as gestures, talismans, cookbooks etc., check out the Key Items and Miscellaneous sheets.

If you're considering starting new game plus, make sure to check out the New Game+ sheet first to see if you've got everything you need.

Configure color themes, import existing profiles and back up your progress in Options.

Q - Toggle the sheets menu

What's included in the walkthrough?

The walkthrough DOES NOT COVER:

  • Any items that require farming, or respawning plants and crafting materials collected from the environment (e.g., Rowa Fruit, Arteria Leafs etc.).
  • Any Scarabs that respawn, such as Crimson and Cerulean Teardrop Scarabs that refill your flasks.
  • Any Spirits or Sword Monuments that players can interact with for lore tidbits.
  • Any Gatekeeper Statues, Birdseye Telescopes, or Stakes of Marika.
  • Any Martyr Effigies that players can interact with to activate Co-op summon signs for that area.

The walkthrough COVERS:

  • Every single item pickup in the game.
  • All achievements, NPCs, questlines, bosses and invasions.
  • All sites of grace, gestures, ashes, talismans, and paintings.
  • All Ash of War, Glintstone, Incantation, and Somber Scarabs.
  • All Scadutree fragments and Revered Spirit Ashes (DLC).
  • While materials collected from the environment aren't included, all Smithing Stones and Gloveworts are.

Do I have to follow every step in order?

Not at all, this project was primarily intended for my cousin and me. Since I am a completionist, I like to make sure that I don't miss any details a game has to offer. The Walkthrough represents how I prefer to complete the game, reducing as much backtracking as possible and approaching the game in a natural, but efficient order.

Elden Ring is a massive game, so if you'd rather play at your own pace, yet don't wanna miss any questlines, try the NPC Walkthrough instead. You can always jump back and forth, checking off tasks in different order, just be mindful that doing some things too early might cancel certain questlines.

Will I get all achievements if I follow the guide?

Yes, all achievements will be naturally achieved during your playthrough if you're following the Walkthrough. No planning needed! If you want to track your achievements and legendary items, check out the Achievements sheet. You can search for a specific achievement by using the search bar in the walkthrough sheet. Achievement-related steps are marked in orange.

Is there anything missable in Elden Ring?

Yes, many things in Elden Ring are missable. While most of them are quest-related, some achievements can be permanently missed until your next playthrough. All missable entries are labeled MISSABLE:, so you can easily filter these by searching missable: in any search bar. Optional or missable entries that don't give any item or reward are labeled (Optional). You won't miss anything if you're following the Walkthrough though.

How long does it take to 100% Elden Ring?

A 100% completion playthrough of Elden Ring following the walkthrough, assuming you achieve all three endings in a single playthrough, will typically take around 60-90 hours. I personally dislike backtracking, so the Walkthrough route is heavily optimized to be as efficient as possible, while still approaching the game in a natural progression order.

What level should I be for each area?

The walkthrough checklists provide recommended level ranges for each area. These ranges don't necessarily represent the difficulty of an area, but rather the level you're likely to reach naturally when following the walkthrough. They are suggestions based on my own playthroughs, and could help you avoid being over- or under-leveled if you want a fairer experience.

I accidentally attacked an NPC, what should I do?

You should never attack or kill NPCs unless it's specifically stated to do so. If you attacked an NPC and they've become hostile, then you can travel to the Church of Vows in Liurnia and absolve your sins at the statue inside by using a Celestial Dew. If you've killed an NPC, then their questline is over until you start your next playthrough or new game plus. Make sure to pick up their Bell Bearing if they were selling items. I also recommend checking the questline of your affected NPC in the Questlines sheet.

How do I follow your directions?

Elden Ring's compass uses cardinal, intercardinal and secondary-intercardinal directions. This means the compass can be broken down into a total of 16 directions:

  1. Cardinal Directions:
    • North (N)
    • East (E)
    • South (S)
    • West (W)
  2. Intercardinal Directions:
    • Northeast (NE)
    • Southeast (SE)
    • Southwest (SW)
    • Northwest (NW)
  3. Secondary Intercardinal Directions:
    • North-Northeast (NNE)
    • East-Northeast (ENE)
    • East-Southeast (ESE)
    • South-Southeast (SSE)
    • South-Southwest (SSW)
    • West-Southwest (WSW)
    • West-Northwest (WNW)
    • North-Northwest (NNW)

For an in-game example, northeast would refer to the middle point between north and east. Using the same logic, north-northeast would then refer to the middle point between north and northeast, while east-northeast would be the middle point between northeast and east. [See visual example]

Can I use this for multiple characters?

Yes! You can create, edit, and delete profiles by using the buttons under profile management in Options. You can also use the NG+ button to reset progress for the Walkthrough, Questlines, Bosses and New Game+ sheets only. Stored information will be saved to your selected profile.

How is my progress saved?

Your progress is stored locally in your browser. This means that clearing your browser's cache will destroy any saved progress.

To back up your data, go to Options and select Backup as file. This will export your data to a JSON file. You can then import your data by selecting Restore from file and the JSON file you exported. If you're on mobile, I recommend using the clipboard options to sync progress between browsers, as JSON files are not recognized as a valid file type on mobile devices.
