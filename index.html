FAQ

How do I use the cheat sheet?

This project is a lightweight collection of checklist sheets, covering everything from single item pickups, NPCs, questlines, bosses, achievements and much more. If this is your first playthrough, I highly recommend that you complete the game on your own first, then use the cheat sheets for your second playthrough instead.

In the Resources sheet, you can find helpful info about soft caps, upgrade tables, rune item values and more.

In the main Walkthrough sheet, you have a step-by-step checklist guide through the entire game, covering as much as possible in just one playthrough.

If you're not interested in a guided approach, but still want to make sure you don't miss any important questlines or bosses, try the Questlines or Bosses sheets instead.

The Achievements sheet can be used to track what achievements and legendary items you're missing.

Similarly, the Armaments, Armor, Ashes, and Spells tabs can be used to track items in their respective categories.

For checklists on miscellaneous items, such as gestures, talismans, crystal tears, cookbooks etc., use the Key Items and Miscellaneous tabs.

If you're considering starting a New Game Plus, make sure to check out the New Game Plus tab to see if you've got everything first.

You can filter any checklist by using the searchbar to only show display items matching your search.

The website automatically assigns a color mode based on your system preference, but you can manually swap between themes by clicking the sun/moon icons.

What's included in the walkthrough?

The walkthrough DOES NOT include:

  • Any items that require farming, respawning plants/crafting materials collected from the environment (e.g. Rowa Fruit, Arteria Leafs etc).
  • Any scarabs that respawn, such as red and blue teardrop scarabs that refill your flask charges.

  • Although ...., smithing stones and gloveworts are included.
